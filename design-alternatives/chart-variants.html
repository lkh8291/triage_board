Reports / Projects 카드 — chart variant 시안

severity는 신뢰도가 낮으므로 보조 정보로만 노출(필요 시 작은 ⚠ 플래그). chart의 주된 시각화는 triage 진행 상태(TP / FP / split / pending)를 담당. 같은 mock 데이터 3개 카드(부분 진행 / 완료 / 신규)에 10가지 변형을 동일하게 적용.

TP (true positive) FP (false positive) Split (의견 분기) Pending (미처리)
V1.

Stacked progress bar + percentage

하나의 bar로 verdict 비율, 우측에 % triaged 큰 숫자. 가장 익숙한 패턴.
store.example.com RPT-2026-04-29-A Web 12 findings · ai-scanner-v0.3
58%/12 ⚠ 2 critical
com.example.shopping RPT-2026-04-29-B Android 8 findings · android-scanner-v0.2
100%/8 ✓ resolved
iot-camera-fleet RPT-2026-04-29-C Device 5 findings · device-scanner-v0.1
0%/5 ⚠ 3 critical
V2.

Pill row (verdict count chips)

각 verdict를 고체 chip으로. 숫자 위주, bar 없음. 가장 데이터 dense.

4 TP 2 FP 1 split 5 pending ⚠ 2 critical
com.example.shopping RPT-2026-04-29-B Android 8 findings
5 TP 3 FP ✓ all resolved

5 pending ⚠ 3 critical
