Reflected XSS in user profile name parameter

found by ai-scanner-v0.3 · 2026-04-29 03:14 UTC · report RPT-2026-04-29-A

Summary

본 취약점은 GET /api/users/profile의 user profile 렌더러에서 name 쿼리 파라미터가 HTML 응답 본문에 unescaped로 반영되는 이유로 reflected XSS가 가능한 취약점입니다. 이로 인해 공격자는 인증된 세션을 가진 피해자가 조작된 URL을 클릭할 경우 피해자의 브라우저에서 임의 JavaScript를 실행해 세션 쿠키 탈취 등을 할 수 있습니다.
